Glossary



1. 404 error - it’s a redirect created for the website to move people from page with doesn’t exist. User which visits website is moved to another URL which provide them information that there is no content on the page they are looking for.



2. 301 redirect - this is permanent option which redirects from the selected page into another. For example, when website is with new page for contact with URL: page.com/contact moves user into new URL such as page.com/contactus, from the old page into new one.



3. Breakpoints -This are the points created for website content to provide the best possible layout to view content. Their job is to make sure design is responsive to different devices such as mobile phones, tablets and computers.



4. Cookies -This is unique file which is saving anonymous user data to give possibility to provide the website information about the user which have connected into page. Cookies are save on the user computer in hard drive folder.



5. CMS Content Management System - that’s a system which is in control of the content on the website. This system let us log in to the backend of the website and edit content such as text and images. CMS is system designed to simplify the usage of the publication webpages.



6. Compression - compression is a progress which is letting us to reduce formats of the images or other web resources. Reduced format of different files improves speed of loading website and also of loading content in user’s computers.



7. CSS Cascading Style Sheets - It’s a scripting language which works next to the HTML. It’s taking care of the performance and visuals of the website. With visuals I mean layout of the page.



8. Browser - It’s an application which is letting us to browse websites and have a preview of them. Browser can open programming languages such as HTML, JavaScript, PHP, CSS and many others.



9. Comment - This is an option in programming language to leave comments at the code without interrupting the code on the website. They are useful for other programmers to find out what code is about.



10. Domain - It’s an address for the website which is letting us access the website. For example, when we want to open google we are typing www.google.com we are entering domain with google website.



11. Favicon- That’s a small icon which which is giving us website’s brand image on the browser tab.



12. Fork - That’s a term which relates to copying of the content from somebody else’s work and using it for their own work such as improve it or change it.



13. FTP (File Transfer Protocol) - It’s a network protocol which is used to transfer files from the computer or any other machine into the server thru internet.



14. GUI (Graphical User Interface) - It’s an interface which is providing us layout of the pages which is later visible on the website by our browser. This include content and different functions on the page.



15. HTML (Hypertext Mark-up Language)- It’s a language which is responsible for the structure of the web page. It’s cooperating with the CSS and it’s responsible for website functionality.



16. Hyperlink-This are the links which are connecting web pages on the website. On the navigation bar we can press on the buttons and move into different web page.



17. JavaScript - It’s a programming language which is used to manage website with their visually effects. There are many options and setting for the websites.



18. JPEG- That’s the most used image format by people It’s keeping quality of the picture also with smaller size of the file. Most of the websites use JPEG for their images.



19. PNG-It’s another image format also with great quality but minus of this format is big size of the file. That’s why it’s not that useful like JPEG.



20. Resolution-It’s about the size of the display on the device and also size of the pixels placed on the screen. Most computers now use resolution such as: 1920 x 1080.



21. Script - Script is mostly used in the code of the website for example in HTML. Scripts are giving us possibility to use different options such as making website interactive and more interesting for people.



22. URL (Uniform Resource Locator) - It’s an address which is letting us to connect to the website on the browser.



23. Template - It’s a template file which is used for the structure of the design for the page or structure of it.



24. Usability - This is a reference to the usage of the page. It’s important for the website to be easy at the operating it by users.



25. Web Page- That’s a document which is visible by people on the browser in type of the website. All the pages are hosted on the World Wide Web.



26. Web Server -It’s a server which is holding website files and make it possible by user to see it on the browser.



27. Wireframe -It’s a design created to give us visual view of the page before creating it.



28. Zip file - It’s a format of the file which is letting us to pack our files into it and let us transfer our Zip folder into another destination much faster than not packed files.



28. www (World Wide Web) It’s a subnet which send all the web pages and documents in the internet over HTTP and HTTPS protocols.



29. Database - It’s a place such as server where we store data such as documents, files and other data which is possible to send by internet into the server or cloud on the internet. Mostly databases are servers which are owned by big companies and secure enough to protect our files.



30. DNS (Domain Name System)- It’s a system which translate domain names into IP addresses which are visible on the browser.



31. Download -It’s a process where data is transferred from the server into our local computer or other device.



32. Ethernet -It’s a type of internet which works after connecting the cable into the computer, name of it is LAN.



33. Frame - That’s area which is a container for other different part such as button or video in the rectangular area or textboxes/images.



34. Forum -It’s a website created for people to talk with others about various subjects.



35.Gateway -It’s a device on a network that works just like another network.



36. Gigabyte - It’s a unit of data which is a size of the 1024 megabytes. It’s a size of the file that also can be stored on the server or user device.



37. Hosting -It’s a service that let us store our files on their servers and access them from the domain on the internet.



38. Link - It’s a URL used by people to send http links to the other pages, we can also link our buttons into different option on the website.



39.Meta Tags -These are HTML tags which are used to describe different things on the webpage such as author, content or picture and many more.



40. MP3 -MP3 it’s a type of file which is giving us possibility to hear audio on the website for example. It’s small format which is easy to transfer into website and server.



41. MySQL -It’s a open source database system which is used to interact with website applications and scripts.



42. Navigation bar-It’s an option on the browser which is letting people navigate the website by pressing on buttons and change pages into another one.



43. Google chrome It’s a browser which is letting us use internet and open websites on browser.



44. Ping -It’s a method which is giving us possibility to check the accessibility to the website by checking if we can connect into IT address of the website.



45.Protocol -These are responsible for following the rules of internet by dictating the programs, computers and also different devices communicate with internet by HTTP and other formats of protocols.



46. Redirect-It’s a script that is forcing our browser to open different page after accessing something with script for redirecting. For example, after selecting something on the internet to buy we are redirected to the payment method.



47. Upload-It’s a process which is letting us to transfer files from local computers to remote locations.



48. Visitor -That’s someone who is accessing the website.



49. VPN -It’s Virtual Private Network that uses encryption and other security options which is creating private network closed for public network users.



50. XHTML -It’s Extensible Hyper Text Mark-up Language which is an future evolution of the HTML.
